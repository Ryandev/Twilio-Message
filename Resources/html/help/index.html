Sign Up

Before using, you will need to create an account on Twilio.com

Logging In

After logging in on Twilio.com, go to your Dashboard->Settings->General and use the 'Account SID' for the username field, the 'Auth token' for the password field during app login

Sending Messages

From: You can set the from field to any alpha-numeric sequence up to 11 characters long (more info)

To: This can be any number worldwide, but it must start with '+' followed by the country code & finally your destination number

Body: Up to 160 characters are allowed in a single message, you can send longer messages however you will be be billed according to the 160 character per message rate (more info)
